# ~/repos - Repository Root

All cloned repositories live under `~/repos/` using the full slug pattern:

```
~/repos/<org|user>/<reponame>
```

Examples:
- `~/repos/scottidler/obsidian` - personal repo
- `~/repos/tatari-tv/philo` - work repo
- `~/repos/danielmiessler/fabric` - third-party repo

## Conventions

- The first directory level is the GitHub org or username
- The second level is the repository name
- This mirrors the GitHub URL structure: `github.com/<org|user>/<reponame>`
- If a repo or tool is mentioned by name, check here before asking where it is

## Identity Context

Repos under `scottidler/` are personal (home identity).
Repos under `tatari-tv/` are work (work identity).
Everything else is third-party or forked.
